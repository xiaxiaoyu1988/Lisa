# lisa
A cross platform ui kit base on [cefpython](https://github.com/cztomczak/cefpython)

- [lisa](#lisa)
    - [Install](#install)
    - [Package](#package)
    - [ToDo](#todo)

## Install
```
    pip install cefpython3==57.0
```

Windows Platform
```
    pip install pywin32
```

## Package
Base on pyinstaller
```
    pip install pyinstaller
```
